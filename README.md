# 🔐 Encriptador de Archivos

Este proyecto permite cifrar y descifrar archivos de texto utilizando diferentes algoritmos de encriptación: **AES-256**, **Base64** y **Cifrado César**. Ofrece una interfaz intuitiva donde el usuario puede seleccionar un archivo, elegir el algoritmo, introducir una clave si es necesario, y obtener el resultado encriptado o desencriptado, con opción de descarga.

## 🚀 Tecnologías utilizadas

- HTML5 + CSS3
- JavaScript (puro)
- [CryptoJS](https://github.com/brix/crypto-js) para el cifrado AES-256

## 🧠 Algoritmos soportados

- AES-256 (requiere clave)
- Base64 (sin clave)
- César (requiere clave numérica)

## 📦 Características

- Soporte para archivos `.txt`, `.json`, `.csv`, `.md`, `.xml`, `.js`, `.html`, `.css`
- Proceso de cifrado y descifrado local en el navegador (sin subir archivos a servidores)
- Descarga directa del archivo resultante
- Interfaz responsiva y accesible
